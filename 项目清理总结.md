## 🚀 使用方法

现在的项目结构很清晰，只需要：

### 1. 确保MySQL服务运行
```cmd
net start mysql
```

### 2. 启动系统
```cmd
# 使用批处理文件（推荐）
start-mysql-server.bat

# 或使用PowerShell脚本
.\start-mysql-server.ps1
```

### 3. 测试数据库
```cmd
# 快速测试
node mysql-quick-test.js

# Web测试页面
http://localhost:3000/mysql-test.html
```
